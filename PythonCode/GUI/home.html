Epilepsy Surgery Planning tool

This tool is for the purpose of aiding in the research of identifying optimal resection sites in epilepsy surgery. In addition this software aims to provide the basis for a tool which could be used by clinicians. Mathematical models evaluate prospective epilepsy surgical resection in silico. A genetic search algorithm is used to exploit this model and identify the optimal nodes to select for resection.
